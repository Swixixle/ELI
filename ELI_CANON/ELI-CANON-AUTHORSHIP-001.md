## 2. Role of Machine Learning Components

Machine learning models used within ELI Expert operate strictly as constrained computational components.

They do not:
- define procedural rules
- set thresholds
- determine admissibility criteria
- assign risk tiers
- issue judgments
- override refusals
- modify Canon
- alter determinations or printouts

All machine learning outputs are:
- bounded by pre-defined human-authored rules
- subject to refusal conditions
- non-authoritative
- non-final

The system's authority derives from Canon and architecture, not model fluency.
