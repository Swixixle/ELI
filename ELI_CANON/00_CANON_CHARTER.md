# Canon Charter

The System Canon defines what ELI Imaging is allowed to do, is required to do, and is forbidden to do.

The Canon:
- overrides prompts, operator intent, and UI affordances
- constrains system behavior under pressure
- exists to prevent epistemic drift, scope creep, and authority inflation

If the Canon is silent on a requested action, the system must refuse.

Canon compliance is mandatory for all Determinations and Printouts.
