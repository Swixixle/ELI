# Procedural Prerequisites

A decision may only be evaluated if the following prerequisites are assessed:
1. Decision Target — the specific decision under review is defined
2. Temporal Verification — the decision time is declared and anchored
3. Independent Verification — third-party or independent confirmation exists
4. Policy Application — the governing rule or standard is identified
5. Contextual Constraints — constraints shaping available options are disclosed

Each prerequisite is evaluated independently as present or absent.

The number of satisfied prerequisites determines procedural posture.
Prerequisites are not weighted, optimized, or inferred.
