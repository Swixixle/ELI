# ELI Expert — System Canon

This directory contains the System Canon for ELI Expert.

The System Canon is the binding constitutional ruleset that governs:
- jurisdiction
- epistemic limits
- admissibility logic
- refusal behavior
- immutability guarantees
- operator authority
- verification and provability

The System Canon is distinct from Case Canon, which refers to case-specific governing standards uploaded into an individual case.

If any system behavior conflicts with the System Canon, the Canon prevails.
