Each target item format

Target Text (full sentence)

[ binary ] [ policy-anchored ]

2) Custom (Advanced)

Intro text (fixed)

Write a decision target as a single sentence that is answerable (Yes/No or Permitted/Not Permitted).

Text area placeholder (fixed)

Example: “Is disciplinary action against [party] procedurally defensible under [policy] based on the documented record?”

Validation errors (fixed, machine tone)

If empty:
	•	DECISION_TARGET_REQUIRED

If not binary (no “is/are/was/can/should/permitted/defensible” etc. heuristic):
	•	DECISION_TARGET_NOT_DECIDABLE

If contains “what/why/how” (heuristic):
	•	DECISION_TARGET_EXPLORATORY_UNSUPPORTED

Display a short help line under errors (fixed):
Decision Targets define the decision. They are not questions to the system.

Buttons
	•	Set Decision Target (primary)
	•	Cancel
