Step 1 — Create ONE shared fetch wrapper

File: client/src/lib/constitutionalFetch.ts

Purpose: every UI call to /evaluate (and later /acks, /determine) goes through a single function that classifies outcomes.

Implementation contract:
	•	403 + error === "CONSTITUTIONAL_REFUSAL" ⇒ return { kind: "refusal", refusal }
